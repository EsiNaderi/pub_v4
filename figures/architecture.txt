                  HIERARCHICAL RESONANT NET (HRN)
                  ================================

Input: SMNIST pixel sequence (B, T=784, M=1)
       1 channel, one pixel per timestep, range [0, 1]
       │
       │  D_0 ∈ ℂ^{N_0 × M}  (input encoder; complex)
       ▼
┌──────────────────────────────────────────────────────────┐
│ LAYER 0  SENSORY POOLS                                    │
│ K_0 × P_0 = 4 × 32 = 128 neurons                          │
│ omega ∈ [0.5, 2.5] rad/step  (high frequency)             │
│ Stuart-Landau dynamics:                                   │
│   q = (1-λ) [exp(iω) z + β z (1-|z|²) - γ z + W_rec s + ηDx] │
│   s = H(|q|² - θ)                                          │
│   z = (1 - κs) q                                           │
│ Block-diagonal W_rec ∈ ℂ^{4 × 32 × 32}                   │
└──────────────────────────────────────────────────────────┘
       │
       │  spikes s_0 ∈ {0, 1}^{N_0}    (R2 audit: spikes only)
       │  D_1 ∈ ℂ^{N_1 × N_0}
       ▼
┌──────────────────────────────────────────────────────────┐
│ LAYER 1  MODE POOLS                                       │
│ K_1 × P_1 = 8 × 32 = 256 neurons                          │
│ omega ∈ [0.10, 1.0] rad/step  (mid frequency)             │
│ Block-diagonal W_rec ∈ ℂ^{8 × 32 × 32}                   │
└──────────────────────────────────────────────────────────┘
       │
       │  spikes s_1
       │  D_out ∈ ℂ^{N_out × N_1}
       ▼
┌──────────────────────────────────────────────────────────┐
│ OUTPUT LAYER  CLASS POOLS                                 │
│ K_out × P_out = 10 × 24 = 240 neurons                     │
│ omega ∈ [0.02, 0.30] rad/step  (slow / memory)            │
│ Block-diagonal W_rec ∈ ℂ^{10 × 24 × 24}                  │
│                                                            │
│ Each pool indexed by class c ∈ {0,...,9}                  │
└──────────────────────────────────────────────────────────┘
       │
       │  spikes s_out  (B, T, N_out)
       │
       │  Tail-window mean spike rate per pool:
       │    pool_rate[b, c] = mean over (last T*tail_frac steps,
       │                                  P_out neurons in pool c) of s_out
       │
       ▼
   ┌──────────────────────────────────────┐
   │ READOUT (per-class pool rate)        │
   │ logits[c] = (pool_rate[c]            │
   │             - mean_c pool_rate)      │
   │             × temperature + bias[c]  │
   │                                       │
   │ (optional: aux_head: linear           │
   │  classifier on tail-mean per-neuron)  │
   └──────────────────────────────────────┘
       │
       ▼
   Logits (B, 10) → softmax → CE loss

ARCHITECTURAL PRINCIPLES:
- Heterogeneous omega per neuron (frequency tiling)
- Block-diagonal recurrence (pool independence)
- Spike-only inter-layer communication (R2)
- Per-class output pools ("each class is a resonant basin")

STABILITY ENGINEERING:
- Amplitude clamp on (u, v) to ±3 prevents runaway
- Detached |z|² in SL term prevents cubic gradient explosion
- Frozen θ, γ, β, λ, κ (only ω, η, D, W_rec are learnable)
- Detached spike feedback (e-prop style) reduces gradient through long
  recurrent chains

COMPLEXITY:
- 226k parameters total (default arch)
- ~11s per BPTT step on CPU (batch=32, T=784)
- Each layer's forward is O(B × T × N) with K-fold pool decomposition
